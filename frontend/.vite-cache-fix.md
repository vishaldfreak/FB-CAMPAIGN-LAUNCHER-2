# Vite Cache Fix

If you get "Outdated Optimize Dep" errors:

1. Stop the dev server (Ctrl+C)

2. Clear Vite cache:
   ```bash
   rm -rf node_modules/.vite
   ```

3. Restart the dev server:
   ```bash
   npm run dev
   ```

Alternatively, you can force re-optimization by adding to vite.config.js:
```js
optimizeDeps: {
  force: true
}
```
Then restart the server.
